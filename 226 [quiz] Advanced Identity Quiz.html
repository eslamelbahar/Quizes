Question 1

A company would like to use their on-premises Microsoft Active Directory to connect to its AWS resources. Which service can it use?

1
2
3
4

Correct Answer

1

Directory Services

Explanation

AWS Directory Service makes it easy for you to setup and run directories in the AWS cloud, or connect your AWS resources with an existing on-premises Microsoft Active Directory.

Question 2

Which of the following services can help you manage multiple AWS accounts?

1
2
3
4

Correct Answer

3

Organizations

Explanation

Organizations helps you to centrally manage billing; control access, compliance, and security; and share resources across your AWS accounts.
